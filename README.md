# 🚀 VisionCore: Real-Time Object Detection Dashboard

VisionCore is a professional-grade Computer Vision application that leverages the **YOLO (You Only Look Once)** architecture to perform high-speed object detection. Built with **Python**, **Ultralytics**, and **Streamlit**, it provides an intuitive interface for real-time inference via webcam or image upload.

## 🛠️ Technical Stack
* **Language:** Python 3.14+
* **AI Model:** YOLOv8m (Medium) - Optimized for accuracy and speed.
* **UI Framework:** Streamlit
* **Computer Vision:** OpenCV (cv2) & PIL
* **Environment:** Virtual Environments (venv)

## ✨ Key Features
- **Real-Time Detection:** Live webcam feed processing with bounding box annotations.
- **Adjustable Confidence:** Sidebar slider to control detection sensitivity (Confidence Threshold).
- **Image Support:** Upload `.jpg` or `.png` files to analyze static data.
- **Optimized Performance:** Pre-configured image resizing (640x480) for stable frame rates on consumer hardware.
